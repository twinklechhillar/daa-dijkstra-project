# daa-dijkstra-project
DAA PROJECT: DIJKSTRA SHORTEST PATH ALGORITHM

This project finds shortest path from a source node to all nodes
in a weighted graph using Dijkstra's Algorithm.

Algorithm Type: Greedy Algorithm

Time Complexity: O(V^2)

Applications:
- Google Maps
- GPS navigation
- Network routing

Input:
Graph is given using adjacency matrix inside Java code.

Output:
Shortest distance from source node to all nodes.
